## numpy函数使用

- ***array()*** :创建numpy数组
- ***reshape()*** : 改变形状
- ***quantile(array, double)*** : 把数据从小到大排列，然后按照输入的浮点数获取分位数或序列
- ***transpose()*** : 数组轴的置换
- ***swapaxes()*** : 数组轴的置换
- ***T*** : 二维矩阵转置
- ***shape*** : 获取数组形状
- **loadtxt** ：读取文本文件
- ***empty()*** ：创建一个指定形状和类型且未初始化的数组
- ***full()*** ：返回一个指定形状，指定类型，和指定数值填充的数组
- ***flipud()*** ：垂直翻转，等效于 ***flip(axis=0)*** 
- ***flipir()*** ：水平翻转，等效于 ***flip(axis=1)*** 
- ***flip()*** ：翻转所有轴
- ***tolist()*** ：numpy转list
